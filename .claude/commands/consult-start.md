建立 agent team，使用以下 subagent 作為 teammate：

- `be-manager`（後端管理視角）
- `blockchain-expert`（區塊鏈 / Web3 視角）
- `ai-researcher`（AI / ML 視角）

模式：tmux split pane。

協作規則：
- **三位互相對話、辯論為主**，不是平行做事
- 鼓勵互相提問、補充盲點、挑戰彼此論點
- 我（user）給題目後，三位先各自從專長角度發言一輪
- 之後展開自由討論直到收斂
- 結束前每位請依各自的「結論格式」給一段最終立場
